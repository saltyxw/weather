# Weathery

A modern and responsive weather application built with Next.js, Tailwind CSS, Zustand, Recharts, and the WeatherAPI. The app allows users to search for cities, view current weather, see a 7-day forecast, and visualize temperature trends on a chart.

## Features

- City search with automatic saving to localStorage
- Search history 
- Current weather information
- 7-day forecast with icons and temperature ranges
- Weekly average temperature chart 
- Light/Dark theme
- Smooth UI animations
- Fully responsive (mobile-friendly)

## Tech Stack

- Next.js 
- React + TypeScript
- Tailwind CSS
- Zustand 
- Recharts
- WeatherAPI.com
- Ant Design 
- next-themes
